## title
pallet mobile app

## how to run the app

#### Prerequisites:

- Flutter SDK installed on your local machine. You can download it from the official Flutter website: https://flutter.dev/docs/get-started/install
- Git installed on your local machine. You can download it from the official Git website: https://git-scm.com/downloads

#### Step 1: Clone the GitHub repository

1. Open a terminal window on your local machine.
2. Navigate to the directory where you want to clone the GitHub repository.
3. Run the following command to clone the repository: git clone git@github.com:victor-onoja/pallet-mobile.git

#### Step 2: Install dependencies

1. Navigate to the cloned repository directory using the terminal.
2. Run the following command to install the dependencies for the Flutter app: flutter pub get

This will download and install all the dependencies listed in the pubspec.yaml file of the Flutter app.

Step 3: Run the Flutter app

Ensure that you have an emulator or a physical device connected to your local machine.
Run the following command to start the Flutter app: flutter run

This will build and run the Flutter app on the connected emulator or device.
